Objetivo do projeto

O jogo "Quiz Challenge" tem como objetivo adquirir conhecimento de programação HTML, CSS e JavaScript aos membros responsáveis pelo desenvolvimento do projeto, estando dentro do conteúdo programático do curso de Programação Web ofertado pela associação Ponto Cidadão, no município de Igarassu-PE. As aulas de programação são ministradas pelo professor Fábio Fraga, Tecnólogo em Sistema da Informação pelo Instituto Federal de Pernambuco (IFPE).
